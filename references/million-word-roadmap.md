# 百万字路线图（执行参考）

## 1. 输入最小集
- 题材
- 一句话脑洞
- 主角目标
- 核心冲突
- 预期结局（可暂时模糊）

## 2. 结构化展开
1. 一句话脑洞 → 三句话卖点（冲突、代价、爽点）
2. 三句话卖点 → 三幕十二节点
3. 十二节点 → 卷级路线（建议 8-12 卷）
4. 卷级路线 → 章级冲刺（每卷 40-80 章）

## 3. 建议规模（可调）
- 总字数：200万 - 500万
- 单章字数：2500 - 4500
- 每卷字数：20万 - 45万
- 每10章一次冲刺复盘

## 4. 每10章冲刺模板
- 本轮目标：
- 必达剧情点：
- 必回收伏笔：
- 风格约束：
- 风险清单：
- 验收标准（是否可进入下轮）：

## 5. 失控处理
- 若主线偏离：回滚到 `novel_plan.md` 最近稳定里程碑。
- 若人物漂移：强制以 `character_tracker.md` 为准重写冲突场景。
- 若风格飘移：立即执行 `/风格提取` + `/风格迁移`。

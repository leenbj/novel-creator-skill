# 📚 小说拆解分析器 (Novel Analyzer)

## 简介

这是一个专业的网文拆解 Skill，能够按照四个核心维度系统性分析小说结构：

1. **🎯 金手指系统** - 主角的特殊能力/系统
2. **⚡ 核心爽点/看点** - 最吸引读者的核心卖点
3. **👤 核心人设** - 主角和重要配角的人物设定
4. **🎣 开篇钩子** - 小说开头的悬念设置

## 激活方式

你可以通过以下方式激活这个 Skill：

```
拆解小说
分析小说
小说结构
拆书
小说解析
/novel-analyze
/analyze-novel
```

## 使用方法

### 基本用法

提供小说章节内容，Skill 会自动分析四个维度：

```
拆解小说
[粘贴第一章内容]
[粘贴第二章内容]
...
```

### 带元数据

```
分析小说
小说名：《我不是戏神》
作者：三九音域
章节数：10章

[粘贴章节内容]
```

### 自定义分析维度

```
拆解小说 --focus golden_finger,character

[粘贴章节内容]
```

可选维度：
- `golden_finger` - 金手指系统
- `core_hook` - 核心爽点
- `character` - 核心人设
- `opening` - 开篇钩子
- `plot` - 剧情结构
- `worldbuilding` - 世界观
- `pacing` - 节奏把控
- `tension` - 悬念张力

## 注意事项

1. **章节数量**：建议一次分析 5-15 章，太多会导致分析不够细致
2. **内容完整**：提供完整的章节内容，不要只提供片段
3. **小说类型**：目前最适合网文/爽文/系统文的拆解
4. **评分标准**：评分基于同类作品的对比，10分代表该维度的顶尖水平
